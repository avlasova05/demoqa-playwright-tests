# DEMOQA-PLAYWRIGHT-TESTS

## Cloning and installing dependencies

after doing git clone https://github.com/avlasova05/demoqa-playwright-tests.git

do **cd demoqa-playwright-tests** and do **npm install**

## To run tests

**npm run test:testbox** - to run Textbox Tests only 

**npm run tests:ui** - to run all the ui tests

**npm run tests:all** - to run all tests

**npm run tests:checkbox** - to run Checkbox Tests only
